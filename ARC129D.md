# ARC129 D - A tale of tragedy

Date: 11-16-2024

[Link to problem](https://atcoder.jp/contests/arc129/tasks/arc129_d)

## Problem Statement

Given a circular array, select any index i and repeatedly do

- A[i] += 2
- A[i + 1] -= 1
- A[i - 1] -= 1

Is it possible to make A equal to zero?
